## Installation
1. Download the "Moon Animator 2.rbxmx" file
2. In Roblox Studio go to "PLUGINS" tab and open the "Plugins Folder"
3. Put the "Moon Animator 2.rbxmx" file inside the Plugins Folder
4. Restart Roblox Studio
5. Enjoy
